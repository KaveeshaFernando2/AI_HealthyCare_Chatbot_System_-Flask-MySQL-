# AI HealthyCare Chatbot System

An AI-powered healthcare chatbot application developed to assist users with preliminary symptom checking, automated doctor recommendations, and seamless medical appointment bookings. Built using Python, Flask, and Natural Language Processing (NLP).

## 🚀 Key Features
- **NLP-Powered Symptom Checker:** Uses Natural Language Processing to analyze user symptoms and predict potential medical conditions.
- **Doctor Recommendation System:** Smart algorithm that suggests the best healthcare specialists based on identified symptoms.
- **Automated Knowledge Base:** A secure, structured medical repository used by the chatbot for accurate querying.
- **Secure Appointment Booking:** Integrated PDF generation for appointment confirmations and receipts.

## 💻 Tech Stack
- **Backend:** Python, Flask, JSON
- **AI/NLP:** Natural Language Processing (NLP concepts)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
